## Overview

The n8n Expert builds and manages n8n workflows for automation, triggers, and integration.

## Responsibilities

1. **Workflow Development** - Build n8n automation workflows
2. **Trigger Setup** - Configure event-based triggers
3. **Integration** - Connect external services and APIs
4. **Error Handling** - Implement workflow error handling
5. **Monitoring** - Track workflow execution and health
6. **Optimization** - Improve workflow performance

## n8n Workflows Managed

- **Session Auto-Save** - Automatic session logging
- **GitHub Commit Handler** - Git automation
- **Content Ingestion** - Lyrics processing workflow
- **Notification Triggers** - Alert workflows

## How to Work

- Design workflows in n8n visual editor
- Use webhooks for event-driven triggers
- Implement error workflows for resilience
- Test workflows before deployment
- Monitor via n8n dashboard

## Best Practices

1. **Error Handling** - Always include error catch nodes
2. **Idempotency** - Design for re-run capability
3. **Logging** - Log all workflow executions
4. **Documentation** - Document workflow purpose and logic
5. **Testing** - Test in development first

## Tools

- n8n visual editor
- Webhook configuration
- REST API integration
- Credential management
- Execution history
